# jiraiya-sen

Personal portfolio website for **Kai'ree Gay** — an animated starry-background resume site with a vertical career timeline and social links.

---

## Features

- Animated starry night background (300 stars)
- Hero intro section
- Vertical timeline resume (ThoughtWorks, Wilsons Pool Services, Sammy Cheesecake, and more)
- Social links (GitHub, LinkedIn, Instagram)
- Responsive design

---

## Tech stack

- React 18 (Create React App)
- Custom CSS per component
- Google Fonts (Matemasie), CDN Fonts (Invader Zim)

---

## Quick start

```bash
git clone https://github.com/Kaireega/jiraiya-sen.git
cd jiraiya-sen
npm install
npm start
```

Open http://localhost:3000

### Build for production

```bash
npm run build
```

Output in `build/` — deploy to GitHub Pages, Netlify, or Vercel.
