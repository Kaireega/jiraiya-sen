# jiraiya-sen

[![React](https://img.shields.io/badge/React-18-61DAFB?logo=react&logoColor=white)](https://reactjs.org/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

Personal portfolio and resume site for **Kai'ree Gay** — animated starry-background UI with a vertical career timeline.

**Live demo:** Run locally with `npm start` → http://localhost:3000

---

## Features

- Animated starry night background (300 stars)
- Hero intro section
- Vertical timeline resume (ThoughtWorks, Wilsons Pool Services, Sammy Cheesecake, and more)
- Social links (GitHub, LinkedIn, Instagram)
- Responsive design

---

## Tech stack

- React 18 (Create React App)
- Custom CSS per component
- Google Fonts (Matemasie), CDN Fonts (Invader Zim)

---

## Quick start

```bash
git clone https://github.com/Kaireega/jiraiya-sen.git
cd jiraiya-sen
npm install
npm start
```

### Build for production

```bash
npm run build
```

Deploy `build/` to GitHub Pages, Netlify, or Vercel.
